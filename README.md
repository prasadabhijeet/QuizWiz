# QuizWiz

QuizWiz is a modern, interactive quiz portal built with Angular. It allows users to participate in quizzes, track their scores, and view leaderboards. The platform is designed for both casual users and administrators who want to create and manage quizzes.

## Features

- User registration and authentication
- Take quizzes with multiple-choice questions
- Real-time score calculation
- Leaderboard to track top performers
- Admin panel for quiz and question management
- Responsive and user-friendly interface

## Getting Started

### Prerequisites
- Node.js (v14+ recommended)
- Angular CLI

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/prasadabhijeet/Angular-Quiz-Portal.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Angular-Quiz-Portal
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   ng serve
   ```
5. Open [http://localhost:4200](http://localhost:4200) in your browser.
